# 📸 샘플 이미지 폴더

이 폴더에 테스트할 시장 가판대 사진을 넣으세요.

## 사용 방법

1. **촬영한 사진 추가**
   - 핸드폰으로 시장 가판대 사진 촬영
   - 이 폴더에 복사 (JPG, PNG, WEBP 지원)

2. **제공된 샘플 사용**
   - 첨부하신 4개 이미지를 여기에 저장:
     - 건강조개류 (800원)
     - 채소 가판대 (오이, 더덕 등)
     - 귤/오렌지 (5000원)
     - 도루묵 (1소쿠리 만원)

3. **테스트 실행**
   ```bash
   python test_ocr.py
   ```

## 📷 좋은 사진 촬영 팁

### ✅ 좋은 예시
- 가격표가 크고 선명하게 보임
- 밝은 조명
- 정면에서 촬영
- 초점이 맞음
- 한 장에 1-3개 가격표

### ❌ 피해야 할 것
- 흐릿한 사진
- 어두운 조명
- 비스듬한 각도
- 가격표가 너무 작음
- 반사광으로 글자가 안 보임

## 🎯 최적 촬영 조건

- **거리**: 가판대에서 50cm~1m
- **해상도**: 최소 800x600 픽셀
- **조명**: 자연광 또는 밝은 실내
- **각도**: 가격표와 수직
- **초점**: 가격표에 정확히 맞춤

## 예상 결과

이미지를 여기에 넣고 OCR을 실행하면:
```json
{
  "products": [
    {
      "product_name": "건강 조개류",
      "price": "800원",
      "unit": "1개"
    }
  ]
}
```

형태로 상품 정보가 추출됩니다!
